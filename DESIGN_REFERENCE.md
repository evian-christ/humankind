# DESIGN_REFERENCE.md

장기 디자인 방향과 아이디어 저장소. 구현 상태나 로드맵은 `SYSTEM_OVERVIEW.md`, 협업 규칙은 `README.md`를 확인하세요.

## 1. 외부 레퍼런스 요약
- **Luck be a Landlord**  
  - 심볼 152개, 평균 상호작용 0.4~0.6개, 핵심 허브 심볼 10~15개.  
  - 인접 시너지, 파괴/생성, 변환, 조건부 배율 등 다양한 메커니즘이 공존.  
  - 문명 테마에 맞춰 유사한 “느슨한 거미줄” 구조를 목표로 하되, 지나친 복잡성은 피한다.

## 2. 우리 게임의 디자인 기둥
1. **아케이드 우선**: 짧은 세션, 즉각적 피드백, 최소 연출로도 명확한 결과.
2. **단일 생존 자원 + 보조 자원**: 식량을 핵심으로 두고, 골드/EXP는 전략 분화를 위한 보조 축.
3. **시대 기반 진행**: 희귀도=시대. 레벨이 오를수록 고대→현대로 확장되며, 선택 가중치가 변함.
4. **명확한 테마 체인**: 농업-산업-도시-과학-전쟁 같은 문명 성장 흐름을 심볼 세트로 표현.
5. **파괴와 진화**: Industrial Revolution, Campfire→City처럼 “파괴/진화가 이득”인 흐름 유지.

## 3. 목표 스케일 가이드
- 단기: 40~60개 심볼로 기본 순환 완성.
- 중기: 허브 심볼 8~10개(예: 정치, 종교, 과학, 경제, 군사) 구축.
- 장기: 80개 이상 + 모드/커스텀 심볼 지원.

## 4. 시스템 확장 방향
### 서브 슬롯(환경 슬롯)
- **구조**: 1×3 패시브 트랙. Spin 시 자동 회전하며 직접 심볼을 추가하지 않고 카운터만 감소.
- **역할**: 적 침략, 자연재해, 황금기, 특수 이벤트 등 “세계 상태”를 예고하는 랜덤 생성기로 사용.
- **예시**: *Barbarian Invasion*은 초기 카운트가 20이며 0이 되면 메인 슬롯에 Barbarian Enemy를 강제 배치.
- **추가 방식**: 시대/레벨 조건 혹은 유물/특수 이벤트로 삽입. 플레이어가 수동으로 구매하는 경우는 예외적.
- **디자인 포인트**: 메인 슬롯이 즉각적 식량/전투를 담당하므로, 서브 슬롯은 장기 전략(언제 위기가 닥치는가, 어떤 보너스를 준비할 것인가)을 고민하게 만드는 요소.

### 유물 시스템
- **획득 루프**: 레벨업 → 심볼 선택 → 유물 선택. 레벨당 1개의 유물을 고르고 영구 보너스를 획득.
- **효과 방향성**: 심볼 유형 보조(예: “전투 심볼 공격력 +1”), 서브 슬롯 조작(카운터 감소/증가), 자원 변환, 선택 페이즈 조작 등.
- **희귀도 체계**: 시대 대신 독립적인 Tier 명칭(Common, Rare, Epic, Legendary 등). 효과 세기/복잡도만 결정.
- **덱 빌드**: 유물이 플레이어의 전략 축을 정의하므로, 서로 보완하거나 상충하는 옵션 세트를 만들어 리플레이 가치를 높인다.

## 5. 아이디어 보관소

### 정치·경제 체계
- **Democracy**: 여러 종류의 심볼이 섞일수록 보너스, 다양성 장려.
- **Fascism**: 한 줄 정렬 시 보너스, 강제 정렬 메커닉.
- **Communism**: 턴 종료 시 모든 심볼 식량을 평균화.
- **Stock Market**: 무작위 심볼 효과를 2배 혹은 0배로 만들며 리스크 제공.
- **Trade Route**: 대각선 위치의 심볼끼리 효과 공유.

### 종교·철학 체인
- **Polytheism**: 서로 다른 종교 수가 많을수록 보너스.
- **Monotheism**: 다른 종교 파괴 시 식량/신앙 스택.
- **Atheism**: 종교가 없을 때 과학 심볼 배율 상승.
- **Philosophy**: 무작위 심볼의 효과를 다른 곳에 복사.
- **Mysticism**: 파괴된 심볼의 효과를 유령처럼 계속 발동.

### 과학·기술 폭발
- **Wheel**: 매턴 보드 회전(슬롯 셔플 변형).
- **Printing Press**: 인접 심볼 효과를 전체 보드에 복제.
- **Gunpowder**: 십자 방향 폭발, 파괴 수만큼 보상.
- **Steam Engine**: 인접 심볼 카운터 속도를 2배로 가속.
- **Internet**: 모든 심볼을 서로 인접한 것으로 간주.

### 문화·예술
- **Music**: 인접 심볼들이 동일한 효과를 공유하도록 동기화.
- **Opera**: 랜덤 “클라이맥스”에 모든 효과 3배.
- **Cinema**: 최근 N턴의 하이라이트를 재상영하여 효과 반복.
- **Video Games**: 전투 미니게임 형태로 보상 경쟁.

### 군사·전쟁
- **Archery**: 멀리 있는 심볼을 저격(직선 공격).
- **Chivalry**: 약한 심볼 대신 피해를 받아 보호.
- **Nuclear Weapons**: 전체 보드 정리 + 막대한 보상 + 방사능 디버프.
