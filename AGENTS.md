# AGENTS.md

이 문서는 저장소 규칙을 최소한으로 요약하고, 세부 안내 문서로 연결하는 포인터 역할만 수행합니다. 실제 온보딩 정보와 시스템 설명은 `README.md` 및 하위 문서를 참고하세요.

## 필수 수칙
1. 사용자 허락 없이는 절대 `git commit` 혹은 `git push`를 수행하지 마세요.
2. 모든 사용자 응답은 반드시 한국어로 작성하세요.

위 두 가지 외의 절차, 워크플로, 시스템 설명은 다음 문서를 순서대로 확인하면 됩니다.

1. `README.md` – 온보딩/규칙/프로젝트 구조/핵심 시스템 개요
2. `SYSTEM_OVERVIEW.md` – 구현 현황, 자원·전투 흐름, 로드맵
3. `DESIGN_REFERENCE.md` – 장기 디자인 목표와 레퍼런스 아이디어

문서 구조가 업데이트되며 과거 내용은 모두 해당 파일들로 통합되었습니다.
