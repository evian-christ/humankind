Humankind — Windows portable (테스트용)

실행 방법
- 압축을 풀고 Humankind.exe 를 더블클릭합니다.
- 별도 설치(인스톨러) 없이 실행됩니다.

환경
- Windows 10/11 권장
- Microsoft Edge WebView2 Runtime 이 필요합니다. (Windows 10/11 에서 대부분 이미 설치되어 있습니다. 없으면 Microsoft 공식 런타임 설치 페이지에서 WebView2 를 설치하세요.)

빌드
- Tauri release 빌드 산출물을 복사한 폴더입니다. (설치용 MSI/NSIS 아님)

버전: 0.0.0
